## 📚简介
coke是一个类springboot的ioc框架，与之不同的是coke更为简单，纯粹。在使用方式上coke继承了springboot的简单易用，在框架本身上对开发者更加友好，是真正拥抱自定义框架的开发者的。

coke仅仅102KB大小，小型项目。由于其在预加载，容器后置处理上采用异步方案，使其相较于springboot项目拥有更高的启动速度。
